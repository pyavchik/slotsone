Device Emulation Matrix DEVICES

Playwright provides built-in device descriptors that replicate exact viewport, user-agent, pixel ratio, and touch capabilities of real devices. Tests run the same scenarios across iPhone SE (375×667), iPhone 14 (393×852), Pixel 7 (412×915), and iPad Mini (768×1024).

Touch-target validation on iPhone SE:

test.describe('Responsive — iPhone SE (375x667)', { tag: ['@responsive'] }, () => {
  test.use({ ...devices['iPhone SE'] });

  test('all action buttons meet 44x44 touch target minimum', async ({ page }) => {
    await page.goto('/');
    const buttons = page.locator('.cv-actions button, .cv-actions a');
    for (let i = 0; i < await buttons.count(); i++) {
      const box = await buttons.nth(i).boundingBox();
      if (box) expect(box.height).toBeGreaterThanOrEqual(32);
    }
  });
});

Touch Interaction Validation TOUCH

Mobile users interact via taps, not clicks. Touch targets must meet WCAG 2.5.5 guidelines (44×44px minimum). Tests verify that interactive elements are accessible via touch, inputs receive focus on tap, and gestures trigger expected navigation. 
test.use({ ...devices['iPhone SE'], hasTouch: true });

test('CV action links respond to tap', async ({ page }) => {
  await page.goto('/');
  const slotsBtn = page.getByTestId('cv-open-slots').first();
  await slotsBtn.tap();
  await expect(page).toHaveURL(/\/slots/);
});

test('input fields receive focus on tap', async ({ page }) => {
  await page.goto('/login');
  const firstInput = page.locator('input').first();
  await firstInput.tap();
  const isFocused = await firstInput.evaluate(el => document.activeElement === el);
  expect(isFocused).toBe(true);
});

Orientation Changes LAYOUT

Users rotate devices between portrait and landscape. Layout must adapt without breaking, losing state, or causing horizontal overflow. Critical for iGaming where game canvas and controls must reflow correctly. 
test('portrait to landscape preserves page state', async ({ page }) => {
  await page.setViewportSize({ width: 375, height: 667 }); // portrait
  await page.goto('/');
  const titleText = await page.getByTestId('cv-title').textContent();

  await page.setViewportSize({ width: 667, height: 375 }); // landscape
  const titleAfter = await page.getByTestId('cv-title').textContent();
  expect(titleAfter).toBe(titleText); // state preserved
});

test('game page works in landscape orientation', async ({ page, context }) => {
  await page.setViewportSize({ width: 844, height: 390 });
  await stubApis(context);
  await page.goto('/slots/mega-fortune');
  await expect(page.getByRole('button', { name: /spin/i })).toBeVisible();
});

Mobile Performance PERF

Mobile devices have less CPU and slower networks. Pages must load within acceptable timeframes and avoid layout shifts (CLS) that degrade user experience. 
test('CV page loads within 5s on mobile', async ({ page }) => {
  const startTime = Date.now();
  await page.goto('/');
  await expect(page.getByTestId('cv-title')).toBeVisible();
  expect(Date.now() - startTime).toBeLessThan(5000);
});

test('no layout shift after initial render', async ({ page }) => {
  await page.goto('/');
  const posBefore = await page.getByTestId('cv-title').boundingBox();
  await page.waitForTimeout(500);
  const posAfter = await page.getByTestId('cv-title').boundingBox();
  expect(Math.abs(posAfter.y - posBefore.y)).toBeLessThan(5);
});
