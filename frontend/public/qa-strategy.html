Release Checklist

  1. 1 All unit tests pass (engine math verified)
  2. 2 API contract tests pass (Zod schema validation)
  3. 3 E2E smoke suite passes (Playwright)
  4. 4 RTP verified within 2% of target via JMeter (10k+ spins)
  5. 5 Paytable cross-referenced with game config
  6. 6 Wallet transaction atomicity confirmed (no balance leaks)
  7. 7 JWT auth flow tested (login, refresh, expiry)
  8. 8 Responsive layout verified (320px–1920px)
  9. 9 Load test shows <200ms p95 under 50 concurrent users
  10. 10 Changelog and test report documented

Key QA Principles

Shift Left

Catch defects early with unit tests and schema validation at the contract level

Risk-Based Testing

Prioritize financial integrity and game fairness over cosmetic issues

Automation First

Automate repeatable checks, reserve manual effort for exploratory sessions

Continuous Feedback

Every PR runs the full test suite; broken builds block deployment
