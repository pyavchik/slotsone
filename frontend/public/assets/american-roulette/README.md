# American Roulette Midjourney Assets

Drop generated Midjourney assets in this directory using the exact filenames listed in:

- `docs/AMERICAN_ROULETTE_ASSET_MANIFEST.md`
- `docs/AMERICAN_ROULETTE_MIDJOURNEY_PROMPTS.md`

The game has automatic CSS hooks for these filenames and will fall back to gradient styling if files are missing.
