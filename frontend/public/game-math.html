Mathematical Verification Approach

  • Theoretical RTP calculated from paytable multipliers multiplied by symbol probabilities on each reel strip
  • Empirical RTP verified via JMeter simulation (17,945+ spins)
  • Chi-squared test applied to symbol distribution to verify RNG uniformity
  • Kolmogorov-Smirnov test for win amount distribution normality
  • Confidence interval: measured RTP 92.46% falls within expected range for sample size (expected convergence to 96% at 1M+ spins)
  • Variance analysis confirms medium volatility classification (SD between 2–5 per unit bet)

Key Takeaways

RTP Accuracy
