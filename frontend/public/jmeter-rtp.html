Bug Discovery

JMeter RTP testing discovered that paytable multipliers were approximately 36x too low across all symbol combinations in the Pyavchik Time Machine slot game.

The actual RTP measured at 2.62% instead of the expected 96.00%. This meant the house edge was 97.38% instead of the intended 4.00% — a critical S1 bug affecting all players.

Test Architecture

1
setUp Thread Group
Auth & Init
Login via POST /auth/login to obtain JWT access token. Initialize game session via POST /game/init to set up wallet and seed pair for the test run.
2
Main Thread Group
RTP Spin Loop
1,000,000 iterations of POST /spin with configurable bet amount. Groovy JSR223 PostProcessor accumulates totals in real time — wagered, won, hit count, scatter count — using BigDecimal precision to avoid floating-point drift.
3
tearDown Thread Group
RTP Report
Groovy script calculates final RTP, hit rate, scatter rate, average win, and max win. Results are logged to JMeter console and written to a summary file for CI pipeline integration.

Methodology

  • Apache JMeter 5.6 used for automated spin execution against the RGS backend
  • JWT authentication with automatic token refresh every 12 minutes to prevent session expiry
  • Groovy JSR223 scripts for real-time data accumulation using BigDecimal precision to eliminate floating-point rounding errors
  • Configurable parameters: spin count, bet amount, request pacing, and target server URL
  • Rate limit handling: non-200 responses tracked separately and excluded from RTP calculations
  • RTP formula: (Total Won / Total Wagered) × 100%
  • Statistical significance: 1M+ spins for reliable RTP measurement, following the Law of Large Numbers to minimize variance

Key Metrics Explained

RTP (Return to Player)
Percentage of wagered money returned to players over time. A 96% RTP means that for every $100 wagered, $96 is returned on average.
House Edge
100% minus RTP — the casino's mathematical advantage. A 4% house edge means the operator retains $4 per $100 wagered on average.
Hit Rate
Percentage of spins that produce any win. A higher hit rate means more frequent payouts, contributing to player engagement.
Scatter Rate
Frequency of bonus or scatter symbol triggers. Scatter symbols typically activate free spins or bonus rounds regardless of payline position.
Volatility
Variance in win amounts over a session. Higher max wins relative to average bet indicate higher volatility — less frequent but larger payouts.
