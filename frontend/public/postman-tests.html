Quick Start

  • Open Postman and import slots-collection.json and dev.env.json.
  • Select the Slots API - Dev environment in the top-right dropdown.
  • Set base_url, then run Auth → Login — jwt_token is saved automatically.
  • Run a single request: open it → Send → check the Test Results tab.
  • Run the full suite: right-click the collection → Run collection.

Newman CLI

newman run slots-collection.json -e dev.env.json

With HTML report:

newman run slots-collection.json -e dev.env.json -r htmlextra --reporter-htmlextra-export report.html
