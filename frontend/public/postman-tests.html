Quick Start

  • Open Postman and import slots-collection.json — all variables are included in the collection.
  • Base URL is pre-configured to https://pyavchik.space. Run Auth → Register first — jwt_token is saved automatically.
  • Run a single request: open it → Send → check the Test Results tab.
  • Run the full suite: right-click the collection → Run collection.

Newman CLI

npx newman run slots-collection.json

With HTML report:

npx newman run slots-collection.json -r htmlextra --reporter-htmlextra-export report.html
